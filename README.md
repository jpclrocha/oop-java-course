* Enum, Compositions
* OOP concepts: Encapsulation, Abstraction, Inheritance and Polymorphism
* Working with Files (File, FileReader, BufferedReader, FileWriter and BufferedWriter)
* Interfaces
* Generics
* Set and Map
* Functional programming and lambda expressions
